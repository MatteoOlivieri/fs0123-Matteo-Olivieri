Come creare un template per i post di Mario Rossi -pubblicato- il 25-10-2020

La prima cosa da fare è suddividere in blocchi logici la pagina, identificando parti comuni da applicare a tutte le pagine

Le varie arie saranno suddivise in:

  • Contenuti d'intestazione del tag (header)
  • Contenuti centrali variabili del tag (main) (corpo principale)
  • Contenuti di fine pagina comuni,da inserire nel tag (footer)
  • Formattazione codice
Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con CSS.Perciò i passi saranno:
  • Progettare il markup assegnando ad ogni specifico testo il tag appropiato
  • Scrivere HTML dai tag esterni a quelli interni dal basso verso l'alto
  • Applicare gli stili CSS
  • L'applicazione di stili CSS è cresciuta negli anni fino a raggiungere la diffusione pressocchè totale
